## Responsive Behavior

### Desktop (>1024px)
- Full sidebar visible
- All tabs visible
- Large upload zone
- Side-by-side result cards

### Tablet (768-1024px)
- Collapsible sidebar
- Scrollable tabs
- Medium upload zone
- Stacked result cards

### Mobile (<768px)
- Hidden sidebar (toggle button)
- Horizontal scrolling tabs
- Compact upload zone
- Single column layout
