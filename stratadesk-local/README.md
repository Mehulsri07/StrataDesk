# StrataDesk - Borewell Visualization

A standalone web application for visualizing and managing borewell strata data.

## Quick Start

1. Open `index.html` in your browser
2. Or double-click `🚀 CLICK_TO_START.bat`

## Features

- Interactive map with location search
- Strata layer visualization
- Excel/PDF data import
- Chart image extraction
- Cross-section view
- PDF report export
- Local storage (no server needed)

## Usage

1. Create a borewell (enter name, coordinates, depth)
2. Add layers manually or import from Excel/PDF
3. View strata chart and cross-sections
4. Export PDF reports

## File Import

Supports:
- Excel files (.xlsx, .xls)
- PDF files with strata data
- Chart images (PNG, JPG)

## Requirements

- Modern web browser (Chrome, Firefox, Edge)
- No installation needed
- No internet required (except for map tiles)
